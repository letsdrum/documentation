Подключение контекстного меню

Необходимо дать доступ кастомному меню

"objects": [
    "mainMenu",
    "customContextMenu1"
]

Далее необходимо описать, что будет находится в контекстном меню при его вызове

{
    "name": "customContextMenu1",
    "title": "Тестовое контекстное меню",
    "hint": "Тестовое контекстное меню",
    "typeMenuItem": "Submenu",
    "submenuItems": [
        {
          "name": "testMenuItem1",
          "title": "Тестовый пункт 1",
          "typeMenuItem": "Method",
          "object": "test_success"
        }
    ]
}

После этого необходимо указать в свойстве menu какой-нибудь таблицы созданное контекстное меню

{
    "name": "menu_test_table",
    "displayName": "Название таблицы",
    "description": "Описание таблицы",
    "fields": [
        {
          "name": "sysstring_field",
          "type": "SysString",
          "displayName": "Тестовое поле"
        }
    ],
    "menus": ["customContextMenu1"]
}

После этого, зайдя в таблицу menu_test_table, нажав правой кнопкой мыши на какую-нибудь запись, получим контекстное с меню со стандартным набором функций: Редактировать и Удалить, а также увидим новую функцию под названием Тестовый пункт 1, которую мы определили в customContextMenu1.
